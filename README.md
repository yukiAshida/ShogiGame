# Travel Expense Sinplification

---
## Overview
---
将棋AI，将棋ゲーム開発用プラットフォーム.

## Requirements
---
* Python 3.6.4
* npm 5.6.0

## Setup
---

パッケージをインストール
```
npm install
```

Flaskをインストール
```bash
pip install flask
```

## Run

サーバーを起動
```
python server.py
```

ブラウザにアクセス
```
https://localhost:8000/
```
